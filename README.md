### Netty使用案例

#### 项目采用Gradle构建.

#### Netty依赖版本4.1.39.Final.

#### 项目结构:
 ```
  ├─src/main/java/com/pjqdyd
      ├─http        netty简单实现http连接服务端
      ├─socket      netty实现socket服务端和客户端
      ├─chat        netty实现多客户连接聊天
      ├─ping        netty实现心跳检测机制
      └─springboot  springboot整合netty
 ```
